# httpserver

The httpserver is simple implementation of an HTTP server . This is personal 
project developed by [Rubens](http://www.rubens-gomes.com).

**This is a work in progress !**

## Installation

- Download and install Apache Maven from:
```
    https://maven.apache.org/
```
- Clone the project from my GitHub:
```
    https://github.com/rubensgomes/httpserver.git
```
## Build, Test and Deploy

- To set up an eclipse project:
```
    mvn -U eclipse:eclipse
```
- To run a java build:
```
    mvn -U install
```
- To run tests:
```
    mvn test
```
- To deploy build:
```
    mvn deploy
```
